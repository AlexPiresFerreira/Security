
# 🏅 Certificações Recomendadas em Segurança da Informação

## 📌 Visão Geral

Este documento apresenta as principais **certificações** em segurança da informação, organizadas por nível de experiência e área de atuação.

🎯 **Objetivo:** Orientar sua jornada de certificações para alavancar sua carreira em segurança cibernética.

---

## 🎯 Certificações por Nível

### 🟢 Entry Level (Iniciante)

#### CompTIA Security+

**Organização:** CompTIA

**Nível:** Iniciante

**Foco:** Fundamentos de segurança

**Tópicos:**
- Ameaças, ataques e vulnerabilidades
- Arquitetura e design de segurança
- Implementação de segurança
- Operações e resposta a incidentes
- Governança, risco e compliance

**Formato do Exame:**
- 90 questões (múltipla escolha + performance-based)
- 90 minutos
- Nota de aprovação: 750/900
- Validade: 3 anos

**Custo:** ~$370 USD

**Preparação:**
- Professor Messer (YouTube - gratuito)
- CompTIA Security+ Study Guide
- TryHackMe: Security+ Path

**Recomendado para:**
- Iniciantes em segurança
- Profissionais de TI migrando para segurança
- Requisito para muitas vagas governamentais (DoD 8570)

---

#### CEH (Certified Ethical Hacker)

**Organização:** EC-Council

**Nível:** Iniciante a Intermediário

**Foco:** Hacking ético e pentest

**Tópicos:**
- Footprinting e reconnaissance
- Scanning de redes
- Enumeração
- Análise de vulnerabilidades
- System hacking
- Malware, sniffing, social engineering
- Hacking web, wireless, mobile
- Cloud computing, IoT

**Formato do Exame:**
- 125 questões (múltipla escolha)
- 4 horas
- Nota de aprovação: 60-85% (varia)
- Validade: 3 anos (ECE credits)

**Custo:** ~$1,199 USD (exame) + curso opcional (~$850)

**Preparação:**
- Curso oficial EC-Council
- CEH v12 Study Guide (Matt Walker)
- TryHackMe / Hack The Box

**Recomendado para:**
- Aspirantes a pentester
- Profissionais que querem entender técnicas de ataque

**⚠️ Críticas:**
- Focado em teoria (múltipla escolha)
- Caro comparado a outras certificações
- Menos respeitado que OSCP na comunidade

---

### 🟡 Intermediate Level (Intermediário)

#### OSCP (Offensive Security Certified Professional)

**Organização:** Offensive Security

**Nível:** Intermediário a Avançado

**Foco:** Penetration testing prático

**Tópicos:**
- Metodologia de pentest
- Enumeração e exploração
- Buffer overflow
- Privilege escalation (Linux e Windows)
- Active Directory attacks
- Port forwarding e tunneling

**Formato do Exame:**
- **23h45min** de exame prático
- Explorar 3 máquinas standalone + 1 AD set
- 70 pontos para passar (de 100)
- Relatório técnico (24h após exame)

**Custo:** $1,649 USD (90 dias de lab + 1 tentativa de exame)

**Preparação:**
- PWK Course (PEN-200) - incluído
- TryHackMe: Offensive Pentesting Path
- Hack The Box: OSCP-like machines
- Proving Grounds (TJnull's list)

**Recomendado para:**
- Pentesters
- Red teamers
- Profissionais que querem certificação prática e respeitada

**💪 Desafio:** "Try Harder" - exame é difícil e exaustivo

**🏆 Reconhecimento:** Uma das certificações mais respeitadas na indústria

---

#### eJPT (eLearnSecurity Junior Penetration Tester)

**Organização:** INE Security (ex-eLearnSecurity)

**Nível:** Iniciante a Intermediário

**Foco:** Pentest básico

**Tópicos:**
- Avaliação e reconhecimento
- Host e network auditing
- Exploração de vulnerabilidades
- Web application security
- Post-exploitation

**Formato do Exame:**
- 48 horas (pode pausar)
- Ambiente de lab prático
- 20 questões (múltipla escolha)
- Nota de aprovação: 70%

**Custo:** $249 USD (curso + exame)

**Preparação:**
- Curso INE (incluído)
- TryHackMe
- PentesterLab

**Recomendado para:**
- Iniciantes que querem certificação prática
- Preparação para OSCP (mais acessível)

**✅ Vantagens:**
- Mais barato que CEH e OSCP
- Prático (não múltipla escolha pura)
- 48h de exame (menos pressão)

---

#### PNPT (Practical Network Penetration Tester)

**Organização:** TCM Security

**Nível:** Intermediário

**Foco:** Pentest de rede e Active Directory

**Tópicos:**
- OSINT
- Network scanning e enumeration
- Exploitation
- Active Directory attacks
- Post-exploitation
- Report writing

**Formato do Exame:**
- 5 dias de pentest em ambiente corporativo simulado
- 2 dias para relatório
- Explorar AD environment completo

**Custo:** $399 USD (curso + exame)

**Preparação:**
- Practical Ethical Hacking Course (TCM Security)
- Windows Privilege Escalation for Beginners
- Linux Privilege Escalation for Beginners

**Recomendado para:**
- Pentesters focados em AD
- Alternativa ao OSCP (mais focado, mais barato)

**✅ Vantagens:**
- Muito prático
- Foco em AD (relevante para empresas)
- Relatório obrigatório (skill importante)

---

### 🟠 Advanced Level (Avançado)

#### OSEP (Offensive Security Experienced Penetration Tester)

**Organização:** Offensive Security

**Nível:** Avançado

**Foco:** Evasão de defesas e técnicas avançadas

**Tópicos:**
- Evasão de AV/EDR
- Process injection e migration
- Bypassing application whitelisting
- Advanced Active Directory attacks
- Lateral movement
- Persistence mechanisms

**Formato do Exame:**
- 48 horas de exame prático
- Relatório técnico (24h após)

**Custo:** $1,649 USD

**Pré-requisito:** Recomendado ter OSCP

**Recomendado para:**
- Pentesters experientes
- Red teamers

---

#### OSWE (Offensive Security Web Expert)

**Organização:** Offensive Security

**Nível:** Avançado

**Foco:** Web application security

**Tópicos:**
- Code review (whitebox testing)
- Exploração de vulnerabilidades web avançadas
- Bypass de proteções
- Desenvolvimento de exploits customizados

**Formato do Exame:**
- 48 horas
- Code review e exploração

**Custo:** $1,649 USD

**Recomendado para:**
- Especialistas em segurança web
- Bug bounty hunters avançados

---

#### OSCE³ (Offensive Security Certified Expert)

**Organização:** Offensive Security

**Nível:** Expert

**Foco:** Certificação guarda-chuva (requer OSCP + OSEP + OSWE)

**Recomendado para:**
- Elite de pentesters
- Demonstrar maestria em offensive security

- Security monitoring

#### GCIH (GIAC Certified Incident Handler)

**Organização:** GIAC (SANS)

**Nível:** Intermediário a Avançado

**Foco:** Incident response e handling

**Tópicos:**
- Incident handling process
- Malware analysis
- Network forensics
- Memory forensics
- Log analysis

**Formato do Exame:**
- 106 questões
- 4 horas
- Nota de aprovação: 73%

**Custo:** ~$2,499 USD (curso SANS) ou $949 USD (apenas exame)

**Recomendado para:**
- Incident responders
- CSIRT members
- SOC managers

---

#### GCIA (GIAC Certified Intrusion Analyst)

**Organização:** GIAC (SANS)

**Nível:** Avançado

**Foco:** Análise de tráfego de rede e detecção de intrusões

**Tópicos:**
- Packet analysis (Wireshark, tcpdump)
- IDS/IPS signatures
- Network forensics
- Protocol analysis
- Threat hunting

**Formato do Exame:**
- 115 questões
- 4 horas
- Nota de aprovação: 73%

**Custo:** ~$2,499 USD (curso) ou $949 USD (exame)

**Recomendado para:**
- Network security analysts
- Threat hunters
- Forensics specialists

---

#### BTL1 (Blue Team Level 1)

**Organização:** Security Blue Team

**Nível:** Iniciante a Intermediário

**Foco:** Fundamentos de blue team

**Tópicos:**
- Security fundamentals
- Phishing analysis
- Threat intelligence
- Digital forensics
- SIEM
- Incident response

**Formato do Exame:**
- 24 horas

**Custo:** $399 USD (curso + exame)

**Recomendado para:**
- Aspirantes a SOC analyst
- Profissionais querendo entrar em blue team

**✅ Vantagens:**
- Prático
- Mais acessível que SANS

### 🟣 Compliance e Governança

#### CISM (Certified Information Security Manager)

**Organização:** ISACA

**Nível:** Gerencial

**Foco:** Gestão de segurança da informação

**Tópicos:**
- Information security governance
- Risk management
- Information security program development
- Incident management

**Formato do Exame:**
- 150 questões
- 4 horas
- Nota de aprovação: 450/800

**Custo:** $575 USD (membro ISACA) ou $760 USD (não-membro)

**Pré-requisito:** 5 anos de experiência em gestão de segurança

**Recomendado para:**
- CISOs
- Security managers
- Risk managers

---

#### CISSP (Certified Information Systems Security Professional)

**Organização:** (ISC)²

**Nível:** Avançado / Gerencial

**Foco:** Segurança da informação (amplo)

**Tópicos (8 domínios):**
1. Security and Risk Management
2. Asset Security
3. Security Architecture and Engineering
4. Communication and Network Security
5. Identity and Access Management
6. Security Assessment and Testing
7. Security Operations
8. Software Development Security

**Formato do Exame:**
- 125-175 questões (CAT - Computerized Adaptive Testing)
- 3 horas
- Nota de aprovação: 700/1000

**Custo:** $749 USD

**Pré-requisito:** 5 anos de experiência (ou 4 anos + graduação)

**Recomendado para:**
- Security architects
- Security consultants
- CISOs
- Profissionais seniores

**🏆 Reconhecimento:** "Gold standard" em segurança (mas criticado por ser muito teórico)

---

## 📊 Roadmap de Certificações

### Path 1: Offensive Security (Red Team)

1. CompTIA Security+ (fundamentos) ↓
2. eJPT (prática básica) ↓
3. OSCP (certificação respeitada) ↓
4. OSEP ou OSWE (especialização) ↓
5. OSCE³ (maestria)


---

### Path 2: Defensive Security (Blue Team)
```

1. CompTIA Security+ (fundamentos) ↓
2. BTL1 (blue team básico) ↓
3. CySA+ (análise de segurança) ↓
4. GCIH ou GCIA (SANS - avançado) ↓
5. CISSP (gerencial)

```
---

### Path 3: Gestão e Governança
```

1. CompTIA Security+ (fundamentos) ↓
2. CISM ou CRISC (gestão de risco) ↓
3. CISSP (certificação sênior) ↓
4. CISA (auditoria) - opcional

```
---

### Path 4: Cloud Security
```

1. CompTIA Security+ (fundamentos) ↓
2. AWS/Azure Cloud Practitioner (fundamentos cloud) ↓
3. AWS Security Specialty ou Azure Security Engineer ↓
4. CCSP (Certified Cloud Security Professional) - opcional

```

## 🎓 Dicas de Preparação

### 1. Crie um Plano de Estudos
```

- Defina meta (ex: OSCP em 6 meses)
- Estude 2h/dia durante semana
- Labs práticos nos finais de semana
- Revise semanalmente

```
---

### 2. Pratique em Labs
```

Certificações práticas (OSCP, eJPT, PNPT):

- TryHackMe
- Hack The Box
- Proving Grounds
- VulnHub

Certificações teóricas (Security+, CEH):

- Questões práticas (Udemy, Pluralsight)
- Flashcards (Anki)

```
---

### 3. Junte-se a Comunidades
```

- Discord: TryHackMe, Hack The Box, Offensive Security
- Reddit: r/oscp, r/AskNetsec, r/cybersecurity
- LinkedIn: Grupos de certificações

```
---

### 4. Documente sua Jornada
```

- Blog pessoal
- GitHub (scripts, notas)
- LinkedIn (compartilhar conquistas)

```
