## 📌 Visão Geral

Lista curada de **cursos online** para aprendizado em segurança da informação, desde iniciante até avançado.

🎯 **Objetivo:** Guiar sua jornada de aprendizado com recursos de qualidade.

## 🆓 Cursos Gratuitos

### 1️⃣ TryHackMe

**Site:** [tryhackme.com](https://tryhackme.com/)

**Tipo:** Plataforma de labs práticos

**Nível:** Iniciante a Avançado

**Paths Recomendados (Gratuitos):**
- **Complete Beginner** (Introdução ao hacking)
- **Pre Security** (Fundamentos de redes e Linux)
- **Introduction to Cyber Security**
- **Web Fundamentals**

**Custo:** Gratuito + Premium ($10/mês)

---

### 2️⃣ Cybrary

**Site:** [cybrary.it](https://www.cybrary.it/)

**Cursos Gratuitos:**
- Introduction to IT & Cybersecurity
- CompTIA Security+ (SY0-601)
- Ethical Hacking
- Network Security
- Incident Response

**Custo:** Gratuito + Premium ($39/mês)

---

### 3️⃣ Professor Messer (YouTube)

**Canal:** [youtube.com/@professormesser](https://www.youtube.com/@professormesser)

**Cursos:**
- CompTIA Security+ (SY0-701) - Completo
- CompTIA Network+ (N10-008)
- CompTIA A+ (220-1101, 220-1102)

**Custo:** 100% Gratuito

**⭐ Recomendação:** Melhor recurso gratuito para CompTIA Security+

---

### 4️⃣ OWASP WebGoat

**Site:** [owasp.org/www-project-webgoat](https://owasp.org/www-project-webgoat/)

**Tipo:** Aplicação web vulnerável para aprendizado

**Foco:** OWASP Top 10, vulnerabilidades web

**Instalação:**

bash docker run -p 8080:8080 -p 9090:9090 webgoat/webgoat

**Custo:** Gratuito (open source)

## 💰 Cursos Pagos (Melhor Custo-Benefício)

### 1️⃣ Udemy

**Site:** [udemy.com](https://www.udemy.com/)

**⚠️ Dica:** Nunca pague preço cheio! Espere promoções (cursos de $100+ caem para $10-15)

**Cursos Recomendados:**

#### Iniciante
- **The Complete Cyber Security Course** (Nathan House)
- **Learn Ethical Hacking From Scratch** (Zaid Sabih)
- **CompTIA Security+ (SY0-601)** (Jason Dion)

#### Intermediário
- **Practical Ethical Hacking** (TCM Security - Heath Adams)
- **The Complete Ethical Hacking Course** (Ermin Kreponic)
- **Web Application Penetration Testing** (Rana Khalil)

#### Avançado
- **Advanced Penetration Testing** (Georgia Weidman)
- **Malware Analysis and Reverse Engineering** (Monnappa K A)

**Custo:** $10-15 (em promoção)

---

### 2️⃣ TCM Security Academy

**Site:** [academy.tcm-sec.com](https://academy.tcm-sec.com/)

**Cursos:**
- **Practical Ethical Hacking** ($30)
- **Windows Privilege Escalation for Beginners** ($30)
- **Linux Privilege Escalation for Beginners** ($30)
- **Open-Source Intelligence (OSINT) Fundamentals** ($30)
- **External Pentest Playbook** ($30)

**Custo:** $30/curso ou $349/ano (todos os cursos)

**⭐ Recomendação:** Excelente para preparação PNPT

---

### 3️⃣ INE Security (ex-eLearnSecurity)

**Site:** [ine.com](https://ine.com/)

**Planos:**
- **Fundamentals:** Gratuito
- **Premium:** $49/mês ou $499/ano
- **Premium Plus:** $99/mês ou $999/ano

**Cursos Inclusos:**
- Penetration Testing Student (PTS)
- Penetration Testing Professional (PTP)
- Web Application Penetration Testing (WAPT)

**Certificação:** eJPT (incluída no Premium)

---

### 4️⃣ Offensive Security

**Site:** [offensive-security.com](https://www.offensive-security.com/)

**Cursos:**
- **PEN-200 (OSCP):** $1,649 (90 dias lab + 1 tentativa exame)
- **PEN-300 (OSEP):** $1,649
- **WEB-300 (OSWE):** $1,649

**Custo:** Alto, mas certificações muito respeitadas

**⭐ Recomendação:** Investimento que vale a pena para carreira

---

### 5️⃣ SANS/GIAC

**Site:** [sans.org](https://www.sans.org/)

**Cursos:**
- SEC401: Security Essentials
- SEC504: Hacker Tools, Techniques, and Incident Handling
- SEC560: Network Penetration Testing and Ethical Hacking

**Custo:** $7,000 - $9,000 (curso + certificação)

**⚠️ Muito caro, mas extremamente respeitado na indústria**

## 🎯 Roadmap de Aprendizado

### Nível 1: Fundamentos (0-3 meses)
```

1. CompTIA Security+ (Professor Messer - YouTube)
2. TryHackMe: Complete Beginner Path
3. Linux Basics (TryHackMe ou Cybrary)
4. Networking Basics (NetworkChuck - YouTube)

```
---

### Nível 2: Prática Inicial (3-6 meses)
```

1. TryHackMe: Offensive Pentesting Path
2. Hack The Box: Easy machines
3. PortSwigger Web Security Academy
4. Udemy: Practical Ethical Hacking (TCM Security)

```
---

### Nível 3: Intermediário (6-12 meses)
```

1. eJPT (INE Security)
2. Hack The Box: Medium machines
3. TryHackMe: Red Teaming Path
4. Bug Bounty (HackerOne, Bugcrowd)

```
---

### Nível 4: Avançado (12+ meses)
```

1. OSCP (Offensive Security)
2. Hack The Box: Hard/Insane machines
3. PNPT (TCM Security)
4. CTFs (DEF CON, Google CTF)

```
