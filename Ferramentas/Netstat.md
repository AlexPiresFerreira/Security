
# 📡 Netstat - Network Statistics

> **Tags:** #netstat #network #connections #troubleshooting #security #monitoring

---

## 📌 Visão Geral

**Netstat** (Network Statistics) é uma ferramenta de linha de comando para exibir **conexões de rede**, **tabelas de roteamento**, **estatísticas de interface** e muito mais.

🎯 **Objetivo:** Monitorar conexões ativas, identificar portas abertas e detectar atividades suspeitas.

---

## 🚀 Sintaxe Básica

bash
##### Linux

netstat [opções]

##### Windows

netstat \[-a\] \[-b\] \[-e\] \[-f\] \[-n\] \[-o\] \[-p proto\] \[-r\] \[-s\] \[-t\] \[-x\] \[-y\] \[interval\]


---

## 🔧 Opções Principais

### Linux

| Opção | Descrição |
|-------|-----------|
| `-a` | Exibe todas as conexões e portas de escuta |
| `-t` | Exibe conexões TCP |
| `-u` | Exibe conexões UDP |
| `-n` | Exibe endereços e portas em formato numérico (sem resolução DNS) |
| `-l` | Exibe apenas portas em estado LISTEN |
| `-p` | Exibe o PID e nome do processo |
| `-r` | Exibe tabela de roteamento |
| `-s` | Exibe estatísticas por protocolo |
| `-c` | Atualização contínua |

---

### Windows

| Opção | Descrição |
|-------|-----------|
| **-a** | Exibe todas as conexões e portas de escuta |
| **-b** | Exibe o executável envolvido na criação de cada conexão |
| **-e** | Exibe estatísticas de Ethernet |
| **-f** | Exibe Nomes de Domínio Totalmente Qualificados (FQDN) |
| **-n** | Exibe endereços e números de porta no formato numérico |
| **-o** | Exibe a ID de processo proprietário (PID) |
| **-p proto** | Mostra conexões do protocolo especificado (TCP, UDP, TCPv6, UDPv6) |
| **-q** | Exibe todas as conexões, portas de escuta e portas TCP não associadas |
| **-r** | Exibe a tabela de roteamento |
| **-s** | Exibe estatísticas por protocolo |
| **-t** | Exibe o estado de descarregamento da conexão atual |
| **-x** | Exibe conexões NetworkDirect, ouvintes e pontos de extremidade |
| **-y** | Exibe o modelo de conexão TCP |
| **interval** | Atualiza estatísticas a cada X segundos |
