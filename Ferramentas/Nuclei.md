
# 🔬 Nuclei - Vulnerability Scanner

> **Tags:** #nuclei #vulnerability-scanner #automation #cve #templates #bug-bounty

---

## 📌 Visão Geral

**Nuclei** é um scanner de vulnerabilidades **rápido e customizável** baseado em **templates YAML**, desenvolvido pela **ProjectDiscovery**.

🎯 **Objetivo:** Automatizar detecção de vulnerabilidades em aplicações web, infraestrutura e cloud.

**Site oficial:** [projectdiscovery.io/nuclei](https://projectdiscovery.io/nuclei)

---

## 📥 Instalação

### Linux (Go)

#### Instalar Go

sudo apt install golang -y

#### Instalar Nuclei

go install -v github.com/projectdiscovery/nuclei/v3/cmd/nuclei@latest

#### Copiar para /usr/bin (opcional)

sudo cp ~/go/bin/nuclei /usr/bin/

#### Verificar instalação

nuclei -version

## 🏷️ Tags Comuns em Templates

Nuclei organiza templates por **tags** para facilitar a busca e execução.

### Tags Principais

| Tag | Descrição | Exemplo |
|-----|-----------|---------|
| **cves** | Templates relacionados a CVEs (Common Vulnerabilities and Exposures) | CVE-2021-44228 (Log4Shell) |
| **fuzz** | Templates que fazem fuzzing em parâmetros ou headers | Teste de inputs aleatórios |
| **exposure** | Informações sensíveis expostas | `.git`, `.env`, backups |
| **misconfiguration** | Erros de configuração | Painéis administrativos abertos |
| **default-login** | Verifica logins padrão em aplicações e dispositivos | admin:admin |
| **panel** | Identificação de painéis administrativos ou interfaces web | phpMyAdmin, cPanel |
| **token-spray** | Testa tokens comuns | API keys, JWTs |
| **ssl** | Verificações relacionadas a SSL/TLS | Certificados expirados, weak ciphers |
| **rce** | Remote Code Execution | Execução de comandos remotos |
| **lfi** | Local File Inclusion | Leitura de arquivos locais |
| **sqli** | SQL Injection | Injeção de SQL |
| **xss** | Cross-Site Scripting | XSS refletido, stored |
| **wordpress** | Vulnerabilidades específicas do WordPress | Plugins vulneráveis |
| **apache** | Vulnerabilidades específicas do Apache | Path traversal |
| **spring** | Vulnerabilidades em aplicações Spring (Java) | Spring4Shell |
| **dns** | Ataques relacionados a DNS | DNS rebinding |
