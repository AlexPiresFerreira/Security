
# 📊 Análise de Logs

> **Tags:** #logs #analysis #forensics #siem #incident-response #bash #awk

---

## 📌 Visão Geral

A **análise de logs** é fundamental para:
- 🔍 Detecção de incidentes de segurança
- 🕵️ Investigação forense
- 📈 Monitoramento de atividades
- 🚨 Identificação de comportamentos anômalos

---

## 🛠️ Comandos Básicos para Análise de Logs

### wc (Word Count)

##### Contar número de linhas

wc -l access.log

#### Contar palavras

wc -w access.log

#### Contar caracteres

wc -c access.log

| Variável | Descrição |
|----------|-----------|
| `$0` | Linha inteira |
| `$1, $2, ...` | Colunas 1, 2, ... |
| `NR` | Número da linha atual |
| `NF` | Número de campos (colunas) na linha |
| `FS` | Field Separator (delimitador) |
