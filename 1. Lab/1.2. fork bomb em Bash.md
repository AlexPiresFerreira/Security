
## O que ela faz

- Cria um processo.
- Esse processo **se replica infinitamente**.
- Cada nova cópia cria outras cópias.
- Em pouco tempo, o sistema fica sem:
    - CPU
    - memória
    - espaço para novos processos

## Para que serve

- Em geral, **para ataque de negação de serviço local**.
- Também é usada em aulas/labs de segurança para mostrar **risco de abuso de shell scripts**.

## O que cada parte da imagem representa

- **`:(){ ... }`**
    
    - Define uma função chamada `:`
    - O nome é só uma escolha “estranha” para confundir
- **`|:`**
    
    - Faz a função chamar ela mesma de novo
    - O `|` indica que a saída é “encaminhada” para outro comando
    - Aqui, a ideia é forçar mais execuções
- **`&`**
    
    - Executa em **background**
    - Isso acelera a multiplicação dos processos
- **`;:`**
    
    - Finaliza a função e logo depois **chama a função novamente**
    - É o gatilho da repetição infinita

## Resumo simples

**É um comando que faz o Bash se chamar repetidamente até esgotar os recursos do sistema.**
